# Student Synthesis Artifacts

This directory contains the controlled student-synthesis layer for
`personlighedspsykologi`.

The source files are older high-performing student notes stored outside this
repository. They are treated as exam-orientation material, not as authoritative
course evidence.

Artifacts:

- `exam_theory_matrix.seed.json` is the normalized curation seed. It may be
  edited deliberately when reviewing the student-synthesis interpretation.
- `source_notes_index.json` is generated by
  `scripts/build_personlighedspsykologi_exam_theory_matrix.py` and stores source
  note provenance, hashes, extraction metadata, and keyword coverage.
- `exam_theory_matrix.json` is generated by the same builder and enriches the
  seed with current course grounding from `course_theory_map.json` and
  `course_concept_graph.json`.

Build:

```bash
./.venv/bin/python scripts/build_personlighedspsykologi_exam_theory_matrix.py
```

Validate existing output:

```bash
./.venv/bin/python scripts/build_personlighedspsykologi_exam_theory_matrix.py --validate-only
```

Do not paste raw student-note extracts into these artifacts. Keep rows compact,
paraphrased, and explicitly grounded against current course artifacts.
