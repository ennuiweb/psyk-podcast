# Personlighedspsykologi Docs

This directory stores show-level planning, source maps, operational runbooks,
and exam-preparation material for `shows/personlighedspsykologi-en`.

## Core Map

- `core/plan.md` - show-level source-of-truth map.
- `core/overblik.md` - compact Danish course overview.
- `data/reading-file-key.md` - canonical lecture-to-reading key used by repo,
  queue, and CI workflows.
- `data/reading-name-sources-report-2026-03-05.md` - provenance note for
  reading names and source cleanup.

## Operations

- `operations/podcast-flow-artifacts.md` - artifact ownership and boundary map.
- `operations/podcast-flow-operations.md` - operational sequence and
  troubleshooting.
- `operations/slides-sync.md` - manual-only slides mapping runbook.
- `operations/preprocessing-system.md` - preprocessing architecture and
  maturity roadmap.

## Learning Materials

- `learning/learning-material-outputs.md` - boundary, quality contract, and
  evaluation map for learner-facing outputs.
- `learning/printout-system.md` - current reading printout contract.
- `learning/problem-driven-printouts.md` - problem-first printout mode.
- `learning/course-understanding-review.md` - course-understanding acceptance
  review.
- `learning/output-generator-codebase-guide.md` - detailed codebase guide for
  the output generator.

## Exam Preparation

- `exam/oral-exam-study-plan-2026.md` - dated oral exam study plan.
- `exam/oral-exam-study-plan-2026-onepage.pdf` - printable one-page A4 reading
  checklist for the 2026-06-03 oral exam, rendered from
  `exam/oral-exam-study-plan-2026-onepage.tex`.
- `exam/oral-exam-printout-priority-narrative.md` - relative priority plan for
  printout reading with narrative psychology as exam center.
- `exam/oral-exam-printout-priority-narrative-onepage.pdf` - one-page
  printable version of the priority plan.
- `exam/oral-exam-printout-generation-plan-narrative.md` - practical generation
  plan for prioritized texts that still lack a complete printout bundle.
- `../exam_priority_config.narrative.json` - priority-plan generator config.
- `../exam_priority_plan.narrative.json` - machine-readable score and bucket
  output.

## Manual Tables

- `data/manual_summary_key.tsv` - manual source mapping table.
- `data/summary_backlog_ocr.tsv` - OCR backlog.
- `data/summary_backlog_readable.tsv` - readable summary backlog.
- `data/summary_backlog_unmapped.tsv` - unmapped summary backlog.
